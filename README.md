# html5-audio-player
A pure HTML5 and JavaScript audio player.


## Getting Started ##

First of all, make sure your page has a div/section with and id
of `player-wrapper`.

```html
...
<div id='player-wrapper'><!-- Your player will appear here. --></div>
...
```
